## Managed Streaming for Kafka
- [Overview](#overview)

### Overview

* AWS `Managed Streaming for kafka` is a streaming data service that manages `apache kafka` infra and operations, making it easier to run `kafka` on aws 
* It provides 3 primary deployment models:
    - `MSK Provisioned`: manually select and size the `ec2 broker` instance types and storage capacity, ideal for predicatable workloads
    - `MSK Provisioned with Express Brokers`: an upgraded provisioned tier built for masssive scale, delivers optimized throughput and faster scaling times
    - `MSK Serverless`: auto provisions, manages, and scales capacity based on real time traffic
