### Overview

* AWS `Amplify` is a complete solution for web and mobile applications
![alt text](images/amplify/image.png)
* It simplifies the process of creating web application infrastructure in the way that beanstalk or lightsail do
    - In the backend it provisions all of these complementary aws services for you, so that all you have to manage is the application code
    ![alt text](images/amplify/image-1.png)
